01 / Decision framework

When this happens, do this.

Seven situations the team will hit. Each has an action, an owner, and one thing not to do.

№ 01
New brief arrives.An incoming request.
ActionPM logs it in Buganizer. Intake via GChat or Monday session. Lead estimates LOE.
PM + VDL

start without a brief and LOE estimate.

№ 02
A task gets blocked.Or risks missing deadline.
ActionFlag in GChat same day. PM re-prioritizes. Lead proposes scope trade-off.
PM + VDL → Matt

wait for the next ceremony to flag it.

№ 03
Creative decision needs approval.A direction call.
ActionVDL reviews first. If client-facing or strategic → Matt Rennick.
VDL · Matt for client

send to Google without Matt's review on Magazine.

№ 04
P0 blocker hits mid-sprint.Something is on fire.
ActionStop current task. PM + lead align on reprioritization. Notify Matt same day.
PM + Lead + Matt

absorb P0s silently — escalate immediately.

№ 05
Work is ready for Google review.Delivery time.
ActionPeer review sign-off required. PM sends via email with sprint report.
VDL sign-off · PM delivers

share with Google before internal peer review.

№ 06
Infinity publish step is completed.Diffusion progress.
ActionMarcela updates the Diffusion Effect publish doc immediately with date, status, and libraries affected.
Marcela owns doc

leave the doc stale — it's Google's source of truth.

№ 07
Scope change proposed mid-project.The brief shifts.
ActionTreat as a new brief. Log it. Flag to PM. Leads validate LOE impact before accepting.
PM + VDL + Matt

absorb scope creep without a new LOE.

02 / Communication

Four channels, distinct uses.

Keep them separate. If a Figma comment matters, it gets a GChat ping. If a decision is final, it goes in email.

№ 01
GChat

Daily. Sync, blockers, quick feedback. FiGS Leads chat for cross-team items. Project-specific chats for each workstream.

№ 02
Figmacomments

Design feedback. Always follow up a batch of comments with a GChat ping so nothing gets lost in threads.

№ 03
Email— formal

Sprint reports. Formal approvals, deliverables, meeting recaps. All weekly reports go out via email.

№ 04
Ceremonies

Keep them all. Monday = FiGS Weekly Ops. VDLs run design syncs. PMs run intake and briefing.
